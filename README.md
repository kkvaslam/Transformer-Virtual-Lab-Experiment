# Transformer Virtual Lab – Realistic Wiring (Part 2: Efficiency vs Load)

This virtual lab replicates the bench setup for a single‑phase transformer efficiency experiment with **interactive wiring** and **live instrument readings**. Students connect the primary/secondary circuits using virtual wires, place ammeters in series, voltmeters in parallel, select load resistance, and record readings.

## Features
- Click‑to‑connect wiring with terminal snapping and correctness validation
- Primary & secondary **voltmeter** and **ammeter** with live readings
- Adjustable **load resistance** (5–280 Ω)
- Non‑ideal transformer model (copper losses + fixed core loss)
- Export data as CSV; optional efficiency plot

## How to Run
1. Open `index.html` locally or host via GitHub Pages.
2. Wire the circuit correctly (see **Help → Show Correct Wiring**).
3. Choose a load resistance and note down Vp, Ip, Vs, Is, Pin, Pout, and efficiency.

## Teacher Options
- In the right panel, you can adjust **Vp**, turns ratio **Ns/Np**, copper resistances, and core loss constant.
